Guía paso a paso para añadir datos

Esta guía muestra el proceso de añadir datos (G.P., M.B.D. y Drg) a la base de datos desde la exportación del archivo de texto de WhatsApp hasta tener los datos totalmente preparados y procesados para, por ejemplo, calcular notas o estadísticas.


Paso 1. Exportar el chat de WhatsApp

Puedes buscar en Google como exportar un chat de WhatsApp a un archivo .txt

Los nombres de contacto que aparecen en el .txt exportado son los mismos que tienes en tu teléfono. Portal G.P. no entiende si tienes añadido a Joaquín como "Joaaaaaa", así que deben configurarse unos alias para su correcta detección (hablar con Anton) o bien reemplazar todos los nombres en el archivo.

Paso 2. Cargar los datos

Dirígete a Importar datos y sigue los pasos para cargar los datos. Una vez hayas subido el archivo, antes de definitivamente cargarlo a la base de datos podrás ver tres tablas de los G.P., M.B.D., y Drg que se han detectado. Ahora es el momento para comprobar si el archivo es correcto. Los datos que no sean válidos (como un falso M.B.D.) serán eliminados después de forma manual. Puedes ver más información sobre las problemáticas de detectar M.B.D. automáticamente aquí.

Para subir los datos definitivamente a la base de datos pulsa "Subir a la base de datos". Este proceso puede llevar algún tiempo según la cantidad de datos. Sé paciente.

Paso 3. Actualizar los ID

Ahora los datos ya están cargados en la base de datos, pero no están conectados entre ellos. En resumen, los G.P., M.B.D. y Drg están conectados entre ellos. Por ejemplo, cada G.P. está conectado a su correspondiente M.B.D y Drg. Los M.B.D. también están conectados a su correspondiente Drg y viceversa. Este proceso puede hacerse de forma automática usando la herramienta Enlazar IDs.

Esta herramienta trata de buscar las correspondencias entre los datos y conectarlos. También tiene algunas limitaciones: si, por ejemplo, para un cierto día en la base de datos hay dos M.B.D. registrados la herramienta Enlazar IDs no sabrá con cuál de los dos enlazar un G.P. y no tomará ninguna acción. Estos casos aparecerán en una notificación amarilla para que luego puedas ir a la base de datos y enlazarlos correctamente.
